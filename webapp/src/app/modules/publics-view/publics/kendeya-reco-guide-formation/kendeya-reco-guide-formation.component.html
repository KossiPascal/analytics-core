Utilisation du Téléphone Tecno Spark 20 et de l'Application KENDEYA

Jour 1

0. Introduction

Ce guide a pour objectif de former les RECO et ASC à utiliser un téléphone Tecno Spark 20 et une application dédiée à la gestion des données communautaires.

Les points suivants seront abordés :

  • Précautions pour optimiser l'utilisation du téléphone et de l'application.
  • Utilisation basique du téléphone.
  • Navigation et manipulation de l'application KENDEYA.
  • Navigation et visualisation des données dans Tableau de bord.

2. Précautions et Conseils

  • Chargez le téléphone avant le début de la journée.
  • Ne laissez pas le téléphone exposé au soleil ou dans des endroits humides.
  • Utilisez exclusivement l'application KENDEYA.
  • Allumer la connexion internet uniquement pour synchroniser les données.
  • Utiliser uniquement les chargeurs fournis pour charger les téléphones afin eviter d'endommager la battérie.

3. Utilisation Basique du Téléphone

Apprenez à manipuler les fonctions essentielles du téléphone :

Allumer/Eteindre le Téléphone

Pour Allumer le téléphone, appuyez sur le bouton d'allumage situé sur le côté droit du téléphone pendant 3 secondes.

Pour Eteindre le téléphone, Appuyez sur le bouton d'allumage situé sur le côté droit du téléphone pendant 3 secondes, puis choisir Eteindre.

Charger le Téléphone

Branchez le câble USB au port situé en bas du téléphone et connectez l'adaptateur à une prise électrique. Vous devez sentir le téléphone vibrer

Déverrouiller l'Écran

Faites glisser votre doigt du bas vers le haut sur l'écran verrouillé, puis saisir le mot de passe.

4. Allumer ou Éteindre la Connexion Internet (3G/4G)

La connexion internet est essentielle pour accéder aux données en ligne dans l’application. Voici les étapes pour activer ou désactiver la 3G/4G :

  1. Accédez au menu déroulant en faisant glisser votre doigt du haut du téléphone vers le bas du coté droit de l’écran.
  2. Appuyez sur l’icône "Données mobiles".
  3. Lorsque l’icône est activée, la connexion internet est fonctionnelle (elle est en bleu).
  4. Pour désactiver, appuyez à nouveau jusqu’à ce que l’icône soit grisée.

5. Activer ou Désactiver la Localisation (GPS)

La localisation est tres nécessaire pour certaines fonctionnalités de l’application. Voici comment activer ou désactiver le GPS :

  1. Accédez au menu déroulant en faisant glisser votre doigt du haut du téléphone vers le bas du coté droit de l’écran.
  2. Appuyez sur l’icône "Localisation" (ou GPS).
  3. Lorsque l’icône est activée, la localisation est fonctionnelle (elle est en bleu).
  4. Pour désactiver, appuyez à nouveau jusqu’à ce que l’icône soit grisée.

6. Diminuer ou Augmenter la Luminosité

Une luminosité optimale améliore l’expérience visuelle et préserve la batterie du téléphone. Voici les étapes pour régler la luminosité :

  1. Accédez au menu déroulant en faisant glisser votre doigt du haut du téléphone vers le bas du coté droit de l’écran.
  2. Identifiez le curseur de luminosité (indiqué par une icône de soleil).
  3. Glissez le curseur vers le haut pour augmenter la luminosité.
  4. Glissez le curseur vers le bas pour la diminuer.

7. Trouver et Ouvrir l'Application

Suivez ces étapes pour accéder à l'application KENDEYA:

  1. Depuis l'écran d'accueil, glissez vers le haut pour afficher toutes les applications disponible dans le téléphone.
  2. Recherchez l'icône suivante : Kendeya avec le logo de l'armoirie de la Guinéé.
  3. Appuyez sur l'icône pour ouvrir l'application.

8. Se Connecter à l'Application KENDEYA

Une fois l'application KENDEYA ouverte, suivez ces étapes pour vous connecter :

  1. Saisissez votre Nom d'utilisateur dans le premier champ.
  2. Entrez votre Mot de passe dans le deuxième champ.
  3. Appuyez sur le bouton "Se connecter" pour accéder à l'application.

Note : Assurez-vous que votre connexion Internet est activée avant de vous connecter.

9. Rafraîchir ou Actualiser l'Application

Pour mettre à jour les données ou résoudre des problèmes de lenteur, vous pouvez actualiser l'application :

  1. Appuyez sur le bouton d'actualisation (flèche circulaire) situé dans le coin supérieur droit de l'écran.
  2. Dans le menu qui apparaît, appuyez sur "Rafraîchir la page".
  3. Attendez que la page se recharge avec les données mises à jour.

Astuce : Utilisez cette fonction régulièrement pour synchroniser les dernières données avec le serveur.

10. Lire et Accepter la Politique de Confidentialité

Après la connexion avec votre nom d'utilisateur et votre mot de passe, il est important de prendre connaissance de la politique de confidentialité :

  1. Une fois connecté, l'application affichera automatiquement la Politique de confidentialité.
  2. Lisez attentivement les informations relatives à la gestion de vos données personnelles.
  3. À la fin du document, cochez la case "J'accepte la politique de confidentialité".
  4. Appuyez sur le bouton "Accepter" pour continuer vers l'écran principal de l'application.

Note : Vous ne pouvez pas accéder à l'application si vous n'acceptez pas la politique de confidentialité.

11. Structure de l'Application : Les Onglets

L'application KENDEYA est organisée en plusieurs onglets pour une navigation facile et rapide :

1. Tâches

Gérez toutes les activités de suivi. Cet onglet permet d'accéder aux formulaires de suivi (suivi des naissances, des visites à domicile, etc.).

2. Historique

Consultez ou modifiez chaque formulaire soumis. Les formulaires sont classés par date et par type d'activité.

3. Contact

Enregistrez de nouveaux ménages et effectuez des consultations sur les membres d'un ménage existant.

4. Analyse

Visualisez des indicateurs clés de performance (ex : taux de suivi prénatal, taux de vaccination, etc.).

12. Utiliser le Menu Supplémentaire

Le menu supplémentaire, accessible via les 3 traits assemblés en haut à gauche de l'écran, offre des options supplémentaires :

  1. Appuyez sur les 3 traits horizontaux situés en haut à gauche.
  2. Le menu suivant apparaîtra avec plusieurs options :
    • Synchroniser les données : Permet d'envoyer ou de recevoir des données mises à jour avec le serveur.
    • Changer son mot de passe : Accédez à une page sécurisée pour modifier vos identifiants.
    • Mettre à jour ses informations : Mettez à jour vos coordonnées et autres détails professionnels.
  3. Sélectionnez une option en appuyant dessus.

Attention : Synchronisez vos données régulièrement pour éviter toute perte d'information.

13. Création de Ménage et du Chef de Ménage

Pour enregistrer un nouveau ménage et son chef :

  1. Ouvrez l’onglet Contact.
  2. Appuyez sur le bouton bleu "+" en bas à droite.
  3. Sélectionnez "Nouveau Ménage".
  4. Remplissez tous les champs requis dans le formulaire, notamment les informations du chef de ménage.
  5. Appuyez sur le bouton "Soumettre" pour sauvegarder les données.

14. Ajouter des Membres du Ménage

Pour ajouter un membre à un ménage existant :

  1. Sélectionnez le ménage souhaité dans l’onglet Contact.
  2. Appuyez sur le bouton bleu "+".
  3. Sélectionnez "Nouvelle Personne".
  4. Remplissez les champs nécessaires dans le formulaire.
  5. Appuyez sur le bouton "Soumettre" pour enregistrer le membre.

15. Effectuer une Consultation sur les personnes

Pour enregistrer une consultation pour un patient :

  1. Dans l’onglet Contact, sélectionnez le ménage correspondant.
  2. Sélectionnez ensuite la personne cible dans la liste des membres du ménage.
  3. Appuyez sur le bouton bleu "+" en bas à droite.
  4. Choisissez le formulaire de consultation approprié (ex : consultation adulte, suivi prénatal, etc.).
  5. Remplissez le formulaire et appuyez sur "Soumettre" pour sauvegarder.

16. Suivi d'Activités Précédemments Effectuées

Pour consulter ou effectuer un suivi d’activité :

  1. Ouvrez l’onglet Tâches.
  2. Sélectionnez le suivi souhaité en fonction de sa période de réalisation.
  3. Les tâches marquées "Aujourd'hui" ou "En retard" sont prioritaires.
  4. Pour chaque tâche, un résumé de la consultation précédente est affiché.

Excercies à faire

Faite tous les formulaires de consultation et suivi des personnes avec vos registres physiques

Rappel des activités du J2

17. Effectuer une Activité Globale dans la Communauté

Pour enregistrer une activité au niveau communautaire :

  1. Sélectionnez un ménage dans l’onglet Contact.
  2. Appuyer sur votre Zone.
  3. Appuyez sur le bouton bleu "+" en bas à droite.
  4. Sélectionnez le formulaire de l'activité souhaitée.
  5. Remplissez le formulaire avec les détails nécessaires.
  6. Appuyez sur "Soumettre" pour enregistrer l'activité.

18. Effectuer des Actions Globales sur un Ménage

Pour exécuter des actions globales au niveau d’un ménage :

  1. Sélectionnez un ménage dans l’onglet Contact.
  2. Appuyez sur le bouton bleu "+" en bas à droite.
  3. Choisissez le formulaire correspondant à l'action souhaitée.
  4. Remplissez le formulaire.
  5. Appuyez sur "Soumettre" pour sauvegarder l'action.

19. Rechercher un Patient avec son Code Unique ou son Nom

Pour rechercher un patient ou un ménage :

  1. Allez dans la barre de recherche située en haut de la zone.
  2. Saisissez le code unique ou le nom du patient ou ménage.
  3. Appuyez sur le bouton Entrée du clavier de votre téléphone.
  4. Les résultats correspondant au code ou nom saisi apparaîtront.

20. Trier les Ménages dans l'Ordre Alphabétique ou par Date de Dernière Visite

Pour appliquer un filtre sur les ménages :

  1. Appuyez sur l'icône de filtre (flèche vers le bas) située en haut à droite, à côté de l'icône de recherche.
  2. Dans la liste déroulante, sélectionnez une des options :
    • Alphabétique: pour trier de A à Z
    • Par date de dernière visite
  3. Le filtre appliquera automatiquement l’ordre choisi.

21. Filtrer Mes Données par Rubrique dans l'Onglet Historique

Pour filtrer vos données dans l’historique :

  1. Ouvrez l’onglet Historique.
  2. Appuyez sur l’icône de filtrage située à droite dans la bande jaune.
  3. Choisissez vos critères de filtre :
    • Type de formulaire (ex : consultation adulte, suivi prénatal, etc.)
    • Date (intervalle ou période spécifique)
  4. Appuyez sur "Valider le filtre" pour appliquer les critères.

22. Modifier un Formulaire Soumis

Pour modifier un formulaire déjà soumis :

  1. Allez dans l’onglet Historique.
  2. Recherchez et sélectionnez le formulaire souhaité.
  3. Appuyez sur les trois points verticaux situés en haut à droite.
  4. Sélectionnez "Modifier" dans le menu déroulant.
  5. Effectuez les modifications nécessaires dans le formulaire.
  6. Appuyez sur le bouton "Soumettre" pour sauvegarder les modifications.

23. Ouvrir le Menu et Naviguer

L'application KENDEYA dispose d'un menu de navigation accessible en haut à gauche :

  1. Appuyez sur l'armoirie située dans le coin supérieur gauche de l'écran.
  2. Le menu s'affiche avec deux options :
    • Application KENDEYA : Permet de revenir à l'accueil principal de l'application.
    • Tableau de bord : Accède aux statistiques et rapports analytiques.
  3. Sélectionnez l'option souhaitée en appuyant dessus.

Conseil : Naviguez entre les sections pour une utilisation optimale de l'application.

24. Visualisation du Tableau de Bord

Le tableau de bord offre une vue d’ensemble des indicateurs clés et des données essentielles. Pour accéder et visualiser le tableau de bord :

  1. Appuyez sur l’icône de menu située dans le coin supérieur gauche (armoirie).
  2. Sélectionnez "TABLEAU DE BORD" dans le menu déroulant.
  3. Les rapports, les tableaux de bord et des pages selon le profile s’afficheront.

Astuce : Utilisez ces informations pour suivre vos performances et identifier les priorités dans votre zone.

25. Filtrer les Rapports par Mois et par RECO

Pour affiner les rapports en fonction de la période ou du relais communautaire (RECO) :

  1. Selectionner le Rapport ou Dashboard souhaité et Accédez à sa page .
  2. Dans les critères de filtre située en haut de la page:
    • Sélectionnez un mois et une année spécifique pour voir les données associées à cette période.
    • Choisissez un RECO dans la liste déroulante pour afficher les données soumises par cet agent.
  3. Appuyez sur "Charger" pour afficher les résultats.
